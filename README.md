# YOLOv8 Object Detection Project

## Overview
YOLOv8 (You Only Look Once Version 8) is a state-of-the-art real-time object detection model developed by Ultralytics. This project demonstrates object detection using YOLOv8 on images, videos, and live webcam feed with high accuracy and speed.

## Features
- Real-time object detection
- High accuracy and fast performance
- Supports image, video, and webcam input
- Easy training on custom datasets
- CPU and GPU support

## Technologies Used
- Python
- YOLOv8 (Ultralytics)
- PyTorch
- OpenCV
- NumPy
